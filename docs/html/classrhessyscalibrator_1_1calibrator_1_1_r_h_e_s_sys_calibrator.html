  basedir,
  worldfiles,
  surfaceFlowtable = False 
)

Ensure that one flow table exists in $BASEDIR/rhessys/flow for each worldfile in the list provided.

Assumes flow tables are named $WORLDFILE_flow_table.dat

Parameters
basedirString representing the basedir from which to read flow tables
worldfilesList of strings representing worldfile names
surfaceFlowtableTrue if surface flowtable is to be used
Returns
Tuple: (True, {}, {}, [], []) or (False, {}, {}, [], []), where the first {} is a dictionary mapping worldfile name to flowtable path, the second {} is a dictionary mapping worldfile name to surface flow table path, the first [] is a list of worldfiles without flow tables, and the second [] is a list of worldfiles without surface flowtables; the lists will be empty if first tuple element is True. The second dict and list will be empty if surfaceFlowtable is False. Flowtable paths will be relative to $BASEDIR/rhessys

OSError if a flow table file is not readable
